Baby Player Settings Guide

When players grow up too quickly, or when we don't want them to grow up, you can adjust these settings in the config file.

💡 Quick Tips:

babyAge = -4000 // If you want faster growth.
babyAge = -2147483647 // If you don't want them to grow up. (Forever Baby)

Full Configuration File (.toml)

[babyplayers_common]
  # The default age for a baby player. Set to 0 to disable baby players.
  babyAge = -2147483647

  # Allows baby players to grow up to adults.
  doAging = true

  # Determines if players respawn as a baby.
  respawnAsBaby = true

  # Determines if baby players can break blocks. 
  # Blocks that break instantly do not count.
  babiesBreakBlocks = true

  # Determines if baby players deal 4 times less damage.
  babiesDealLessDamage = true

  # If set to true, babies can't use ranged weapons like Bows, etc.
  babiesCantUseBows = true
